# ExpertEase
1.Clone the repository using : git clone https://github.com/sakshiashtekar/ExpertEase.git
2.Open the cloned repository in VScode.
3.Open the terminal/cmd and enter command : npm install -> node_modules folder will be installed and should now appear in your folder structure.
4.Pre-requisites to run the project : Install latest expo go application on your android device.
5.To run the project execute : npx expo start or expo start or npx expo
6.After the above command, scan the QR using Expo go app on your android device.
